# 🏥 Clínica Viver Bem - Website Institucional

Este é o site institucional oficial da **Clínica Viver Bem**, desenvolvido com foco em experiência do usuário (UX), design moderno, responsividade e interatividade. O projeto foi estruturado para fornecer informações claras sobre o corpo clínico, especialidades e facilitar o agendamento de consultas.

---

## 🚀 Demonstração

O site está hospedado e disponível para acesso público no Render através do link:
👉 ** https://clinica-viver-bem.onrender.com  **

---

## ✨ Recursos Implementados

O projeto conta com os seguintes recursos e funcionalidades dinâmicas:

*   **Design Responsivo Premium:** Layout adaptável para smartphones, tablets, notebooks e desktops.
*   **Filtro Dinâmico de Especialidades:** Sistema baseado em JavaScript que filtra o corpo clínico na tela em tempo real, sem necessidade de recarregar a página.
*   **Agenda Avançada com Bloqueio Dinâmico:** Formulário inteligente de agendamento que impede a seleção de datas retroativas e adapta os horários disponíveis de acordo com o dia escolhido (com tratamento especial para finais de semana e domingos fechados).
*   **Segurança e Conformidade (LGPD):** Banner explícito de consentimento de cookies e modal integrado detalhando as diretrizes de privacidade dos dados coletados do paciente.
*   **Validação de Formulários com Máscara:** Entrada de telefone celular formatada automaticamente enquanto o usuário digita no padrão `(XX) XXXXX-XXXX`.
*   **Carrossel de Avaliações e Lightbox:** Sistema interativo de feedbacks com transição automatizada e galeria de fotos da estrutura da clínica com efeito de zoom em tela cheia (Lightbox).
*   **Scroll Reveal Integrado:** Animações sutis e seguras de surgimento de conteúdo conforme o usuário rola a página.
*   **Botão Flutuante do WhatsApp:** Atalho rápido posicionado estrategicamente para o paciente iniciar uma conversa direta com a recepção.

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica e acessível de todo o conteúdo.
*   **CSS3:** Estilização moderna utilizando CSS Grid, Flexbox, efeitos de sobreposição (*overlay*) e animações de transição.
*   **JavaScript (Vanilla JS):** Lógica nativa para manipulação do DOM, filtros de busca, regras da agenda, carrossel e modais.
*   **FontAwesome (v6.4.0):** Biblioteca completa de ícones vetoriais modernos.
*   **Google Maps API (Embed):** Mapa de localização geográfica da clínica integrado na seção de contatos.
